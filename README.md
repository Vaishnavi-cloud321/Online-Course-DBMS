# Online Course Registration System

A DBMS project developed using MySQL Command Prompt to manage students, courses, and registrations.

## Features
- Stores student details
- Stores course details
- Manages course registrations
- Allows retrieval of student enrollment and course information using SQL queries

## Technologies Used
- MySQL
- SQL
- Git & GitHub

## Database Structure
- Students Table
- Courses Table
- Registration Table (with foreign key relationships)

## Sample Queries
- View all students
- Find courses registered by a student
- Count number of students in each course
